In BHN data lake we store data in S3 in various format such as Raw data format, Parquet compressed format and Flattened JSON structured data at multiple stages in Pipeline.

For interactive analysis, In BHN Data lake you can create a Glue catalog table and start querying the data using Athena from within the AWS Console.
